# Folder Gambar Produk

Folder ini digunakan untuk menyimpan gambar-gambar produk yang di-upload melalui sistem.

## Format File yang Diizinkan:
- JPG / JPEG
- PNG
- GIF

## Ukuran Maksimal:
- 5 MB per file

## Cara Upload Gambar Produk:

### Melalui Admin Panel:
1. Login sebagai admin
2. Masuk ke menu **Produk**
3. Klik tombol **Tambah Produk** atau **Edit** pada produk yang ada
4. Pilih gambar dari komputer Anda
5. Klik **Simpan**

### Catatan:
- Gambar akan otomatis diberi nama unik untuk menghindari duplikasi
- Gambar lama akan dihapus saat gambar produk diupdate
- Pastikan gambar memiliki kualitas yang baik untuk tampilan yang optimal
- Rekomendasi ukuran gambar: 800x800 pixels

## Struktur Penamaan File:
Format: `[unique_id]_[timestamp].[extension]`
Contoh: `5f8b9a1c2d3e4f5_1234567890.jpg`
